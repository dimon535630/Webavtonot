# Магазин EXE-файлов

Готовый простой сайт на HTML/CSS/JS для продажи ваших `.exe` файлов.

## Что умеет

- Витрина для покупателей с товарами.
- Покупка товара и мгновенное скачивание прикрепленного EXE-файла.
- Админ-панель:
  - управление товарами (добавить/удалить),
  - прикрепление EXE-файла к каждому товару,
  - раздел «Прибыль» с историей покупок,
  - смена пароля админа.
- Обратная связь от покупателей.
- Все сохраняется в браузере через `localStorage`.

## Как запустить

Просто откройте `index.html` в браузере.

Или поднимите локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Вход в админку

- Нажмите «Админ-панель».
- Пароль по умолчанию: `admin123`.

## Важно про загрузку файлов

- Файлы прикрепляются в форме добавления товара (`EXE-файл для скачивания`).
- Так как это демо без сервера, файл хранится в `localStorage` браузера.
- Из-за этого есть ограничение по размеру (рекомендуется до ~4 MB на файл).
- Для настоящих больших файлов и реальных продаж нужен backend + облачное хранилище (S3/Cloudflare R2/и т.д.).
